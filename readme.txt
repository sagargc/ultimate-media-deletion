Comprehensive solution for deleting WordPress media including standard attachments, ACF fields, and HTML content images.

== Description ==

This plugin provides complete media deletion capabilities for WordPress, including:

* Standard post attachments
* ACF Pro fields (image, gallery, flexible content)
* HTML content images
* Bulk deletion with media
* Detailed deletion reports

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultimate-media-deletion` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the bulk actions or delete posts normally - media will be deleted automatically
