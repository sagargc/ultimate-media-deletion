Delete all media associated with posts including ACF fields and HTML content.

== Description ==

Ultimate Media Deletion provides comprehensive media deletion capabilities for WordPress:

* Delete standard post attachments
* Delete media in ACF Pro fields (all types including flexible content)
* Delete images embedded in HTML content
* Admin interface enhancements with warnings
* Bulk deletion actions

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultimate-media-deletion` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the bulk actions or delete posts normally - media will be deleted automatically

== Frequently Asked Questions ==

= Does this work with ACF Pro? =

Yes, it works with all ACF field types including flexible content and repeaters.

= Can I exclude certain media from deletion? =

Not in the current version, but this is planned for a future release.
